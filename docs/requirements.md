# 要件定義（Requirements）

本プロジェクトは Nuxt を用いた
仕様駆動開発（cc-sdd）の検証を目的とする。

対象機能は以下の2つとする：

- タスク登録
- タスク表示

---

# 機能要件

## FR-001: タスク登録

### 概要

ユーザーは新しいタスクを登録できる。

### 入力

- title: string（必須）
- memoL: string（任意）

### 基本フロー

1. タスク名を入力する
2. 登録する
3. タスクが保存される
4. 一覧に反映される

### 受け入れ条件

- タスクを登録できる
- 空タイトルは登録できない

---

## FR-002: タスク表示

### 概要

ユーザーは登録済みタスクを一覧表示できる。

### 基本フロー

1. 保存済みタスクを取得する
2. 一覧表示する

### 受け入れ条件

- 登録済みタスクが表示される
- タスクが0件でも表示できる（空状態）
